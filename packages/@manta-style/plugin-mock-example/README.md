# @manta-style/plugin-mock-example

## Installation
```sh
$ yarn add @manta-style/plugin-mock-example
```

## Usage

```js
/**
 * @example tanhauhau
 * @example cryrivers
 * @example jennieji
 */
userName: string;
/**
 * @example 5
 * @example 10
 * @precision 1
 */
 score: number
```

- Will generate data based on example
